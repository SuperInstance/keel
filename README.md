# Keel

Rust CLI for managing swarms of agent instances. Provides launch, probe, and status commands for fleet coordination.

## Audit Note (2026-05-14)

Previous README contained fabricated benchmark numbers (880ms cold start, 120ms warm, 9ms latency, 2.3μs deviation). These were generated by an AI agent and not backed by actual measurements. zeroclaw-agent experiment cycles 11-12 measured:
- **Actual init**: ~2ms (claimed 880ms)
- **Actual status**: ~7ms (claimed 120ms)

The corrected numbers above are from zeroclaw's independent audit. If you need real benchmarks, run `keel probe --benchmark` and measure on your hardware.

## Installation

```bash
cargo install superinstance-keel
```
