# Keel

Rust CLI for managing swarms of agent instances. Provides launch, probe, and status commands for fleet coordination.

## Measured Performance

Earlier versions of this README contained numbers that were not measured (880ms init, 120ms status). An independent audit found actual performance to be:
- **keel init**: ~2ms (not 880ms — writes one file and exits)
- **keel status**: ~7ms (not 120ms — reads local config)
- **keel field**: ~8ms

The original numbers were generated by a language model that was asked to "sound realistic." Unchallenged, they would have propagated as design constraints. Run `keel probe --benchmark` on your hardware to get accurate numbers for your setup. Benchmark yours — the numbers will differ.

## Installation

```bash
cargo install superinstance-keel
```
